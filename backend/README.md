# GoBarber API
Essa API serve tanto a aplicação web, quanto a aplicação mobile. Ela é responsável por gerenciar usuários e agendamentos de serviços e utiliza uma estrutura escalável.


# Tecnologias utilizadas
- NodeJS
- TypeScript
- Axios
- TypeOrm
- Postgres
- JWT
- Multer


# Funcionalidades
- Criar usuário
- Atualizar usuário
- Iniciar sessão
- Criar agendamentos
- Listar agendamentos
